## Availability Prediction

### Likely Available
- Unique portmanteau combinations (Pipular, FinTopus)
- Uncommon phonetic respellings (Tradr, Sticka)
- 3+ word combinations (NoobTrader, LevelUpFunds)
- Industry jargon combinations (GreenWick, PipsTicks)
- Invented words with no meaning

### Possibly Available (Check Required)
- Common suffix patterns (Tradely, Fundio)
- Short respellings (Clevr, Kwik)
- Popular technique combinations
- Names with numbers

### Likely Taken
- Single dictionary words
- Common two-word combinations
- Popular brand patterns (anything + ly/ify)
- Short names (< 5 characters)
- Common industry terms alone

## Domain Strategy Tips

### Prefix Strategies
If `name.com` is taken, try:
- **get**name.com (GetResponse pattern)
- **try**name.com (TryHoneycode pattern)
- **use**name.com (UseMotion pattern)
- **go**name.com (GoFundMe pattern)
- **my**name.com (MyFitnessPal pattern)
- **hey**name.com (HeyGen pattern)

### Suffix Strategies
If `name.com` is taken, try:
- name**app**.com (Superhuman pattern)
- name**hq**.com (LinearHQ pattern)
- name**io**.com (ClickUp pattern)
- name**now**.com (ServiceNow pattern)
- name**ai**.com (Jasper pattern)

### Alternative TLD Strategy
Priority order if .com taken:
1. **.io** - Most accepted tech alternative
2. **.co** - Short, professional
3. **.app** - If it's an application
4. **.ai** - If AI-related
5. **.dev** - If developer-focused

## Social Handle Availability

### Check These Platforms
- Twitter/X: @name
- Instagram: @name
- LinkedIn: /company/name
- Facebook: /name
- TikTok: @name
- YouTube: /c/name
- GitHub: /name

### Handle Strategy
If exact handle taken:
- Add **app** suffix: @nameapp
- Add **hq** suffix: @namehq
- Add **official**: @officialname
- Use underscore: @name_app
- Add industry: @nametrade

### Tools for Social Check
- Namechk (namechk.com) - Multi-platform
- KnowEm (knowem.com) - 500+ platforms
- Namecheckr (namecheckr.com) - Quick check

## Trademark Considerations

### Before Finalizing
1. **USPTO Search**: uspto.gov/trademarks
2. **Google Search**: "[name] trademark"
3. **Industry Search**: Check competitors
4. **International**: WIPO Global Brand Database

### Red Flags
- Existing trademark in same industry
- Similar phonetic sound to major brand
- Could cause consumer confusion
- Generic industry term (harder to protect)

### Safe Patterns
- Invented words (Pipular, Vodu)
- Unique combinations
- Non-descriptive names
- Names with distinct spelling

## Quick Domain Checklist

### Before Committing to a Name

- [ ] Check .com availability
- [ ] Check alternative TLDs (.io, .co)
- [ ] Verify social handles available
- [ ] Search trademark database
- [ ] Google the exact name
- [ ] Check for similar brands in industry
- [ ] Verify pronunciation works globally
- [ ] Test spelling with friends

### After Choosing

- [ ] Register .com (or best available)
- [ ] Consider buying alternatives (.io, .co)
- [ ] Secure social handles
- [ ] Consider trademark filing
- [ ] Set up domain privacy
- [ ] Configure email forwarding

## Domain Pricing Tips

### Best Time to Buy
- During sales (Black Friday, Cyber Monday)
- New TLD launches (first year discounts)
- Registrar promotions

### Watch Out For
- First year discounts with high renewals
- Premium pricing on "desirable" names
- Add-on fees (privacy, email)
- Transfer restrictions
